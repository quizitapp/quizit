# QuizIt

### <u>Global Settings</u> 

1. Load env vars:
`source load_env.sh`
<br/>

2. Run the following command to setup the local DB
`docker compose up -d`
<br/>


### <u>Run Server</u> 

1. Cd to server server's folder
`cd server`
<br/>

2. Install deps:
`npm install`
<br/>

2. Run server:
`npm run start:dev`
<br/>
